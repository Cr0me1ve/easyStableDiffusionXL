## Installation

## You need to install [python > 3.9](https://www.python.org/downloads/) first

### Installation on Windows 10/11 with NVidia-GPUs
1. [Download](https://github.com/Cr0me1ve/easyStableDiffusionXL/archive/refs/heads/main.zip) or click button "Download ZIP"
2. Run `install.bat`.
3. Run `run.bat`.
